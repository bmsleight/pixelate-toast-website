Printrbot

Tags openscad / printrbot

I now have a wonderful 3d-printer (again), a Printrbot with a little help from replicat3d

It printed in ~50mins. It actually finished the print looking like below, all I need to do is clear up the support material.

I took about the same time to design it in Openscad

$fn=60;
radius_tr = 6/2;
connector_offset = 20;
//
co = connector_offset;
// m6 Width Across Corners 11.05
module connector()
{
    d = (radius_tr*2);
    unit = (d*2); //12
    co_h = unit*3;
    cu_h = unit*3;
    difference() {
        translate([-unit/2,-unit/2,-unit/2]) cube([co_h,co_h,co_h], center=true);
        translate([0,0,0]) cube([co_h,co_h,co_h], center=true);
        translate([unit/2,unit/2,-co_h]) cylinder(h=co_h, r=radius_tr, center=center);
        rotate([0,270,0]) translate([unit/2,unit/2,0]) cylinder(h=co_h, r=radius_tr, center=center);
        rotate([90,0,0]) translate([unit/2,unit/2,0]) cylinder(h=co_h, r=radius_tr, center=center);
    }
}
connector();
